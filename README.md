# PICC 企财险知识库

> 让保险条款像代码一样可读、可标注、可探索 —— 用 AI 将中国人民财产保险股份有限公司企财险条款转化为可交互的知识图谱。

**在线阅读**: [https://111wukong.github.io/picc-property-insurance-kb](https://111wukong.github.io/picc-property-insurance-kb)

---

## 这是什么？

**一句话介绍**：将 PICC 企财险 32 个产品的条款正文（23万+字符）转化为结构化、可标注、可探索的知识库。

**核心成果**：
- **32 个险种产品**，涵盖财产基本险、综合险、一切险及各类特殊险种
- **235,690 字符条款正文**，全部提取为纯文本 + tagged.md 标注格式
- **23 类实体标注体系**（8类条款结构 + 10类核心概念 + 5类合同主体）
- **6 种关系类型**（包含/引用/互斥/扩展/版本/触发）

**适合谁？**
- 📖 **保险从业者**：快速查阅条款、理解责任边界
- 🎓 **保险学习者**：系统化理解企财险产品体系
- 🤖 **AI 工程师**：保险领域知识工程方法论参考

### 核心功能

- **23 类实体语法高亮**：8 类条款结构 + 10 类保险核心概念 + 5 类合同主体，按色标区分
- **实体标注跳转**：点击标注实体可跳转到对应词条页
- **知识图谱**：险种包含关系、版本演进、责任递进网络
- **条款对比**：并排对比不同险种的责任范围差异

## 标注体系

### 条款结构实体（8类）

| 标记 | 类型 | 示例 |
|------|------|------|
| `@` | 险种名 | 财产基本险（2025版） |
| `§` | 条款章节 | 总则、保险责任、责任免除 |
| `¶` | 具体条目 | 第三条、第八条第（二）款 |
| `№` | 备案号 | 人保(备案)[2025]Nxxx号 |
| `+` | 附加险 | 盗窃险附加条款 |
| `≡` | 特别约定 | 免赔额特别约定 |
| `↗` | 版本引用 | 2009版 → 2025版 |

### 保险核心概念（10类）

| 标记 | 类型 | 标记 | 类型 |
|------|------|------|------|
| `☐` | 保险标的 | `⚡` | 保险责任 |
| `✕` | 责任免除 | `$` | 保险金额 |
| `%` | 免赔额/率 | `⏱` | 保险期间 |
| `¥` | 费率 | `⛓` | 赔偿处理 |
| `◈` | 义务条款 | `⚖` | 定义条款 |

### 合同主体（5类）

| 标记 | 类型 | 标记 | 类型 |
|------|------|------|------|
| `▶` | 投保人 | `◀` | 保险人 |
| `◆` | 被保险人 | `📋` | 合同单据 |
| `▣` | 第三方 | | |

## 项目结构

```
picc-property-insurance-kb/
├── corpus/                  # 原始条款纯文本（32个险种）
├── tagged/                  # AI/手工标注后的 tagged.md
├── ontology/                # 保险本体定义
│   └── entity-types.json    # 23类实体 + 6类关系
├── render/                  # 渲染引擎
│   └── render_html.py       # tagged.md → HTML（借鉴 shiji-kb）
├── docs/                    # 发布的站点（GitHub Pages）
│   ├── index.html           # 首页
│   └── chapters/            # 已渲染的章节
└── scripts/                 # 处理脚本
    └── extract_clauses.py   # .docx → corpus/
```
