# PICC企财险知识库 — AI标注指南

## 任务
对企财险条款正文进行实体标注，生成 tagged.md 文件。

## 标注语法
```
〖MARKER 显示文本〗  或  〖MARKER 显示文本|规范名〗（消歧用）
```

## 22类实体标注体系

### 条款结构（7类）
| 标记 | 类型 | 标注对象 |
|------|------|----------|
| `@` | 险种名 | 保险产品全称 |
| `§` | 章节标题 | 总则、保险责任、责任免除... |
| `¶` | 条款条目 | 第一条、第三条第(二)款 |
| `+` | 附加险 | 附加险条款名称 |
| `≡` | 特别约定 | 特别约定内容 |
| `↗` | 版本引用 | 版本号引用 |

### 保险核心概念（10类）
| 标记 | 类型 | 标注对象 |
|------|------|----------|
| `☐` | 保险标的 | 建筑物、机器设备、存货... |
| `⚡` | 保险责任 | 火灾、爆炸、雷击... |
| `✕` | 责任免除 | 地震、战争、核辐射... |
| `$` | 保险金额/价值 | 重置价值、账面余额、市场价值 |
| `%` | 免赔额/率 | 绝对免赔额500元、5% |
| `⏱` | 保险期间 | 一年、以保险单载明为准 |
| `¥` | 费率 | 基准费率、调整系数 |
| `⛓` | 赔偿处理 | 比例赔偿、第一危险赔偿、货币赔偿 |
| `◈` | 义务条款 | 如实告知、及时通知、施救义务 |
| `⚖` | 定义条款 | 重大过失、恐怖活动、重置价值 |

### 合同主体（5类）
| 标记 | 类型 | 标注对象 |
|------|------|----------|
| `▶` | 投保人 | 财产所有人、共有人、承租人 |
| `◀` | 保险人 | 中国人民财产保险股份有限公司 |
| `◆` | 被保险人 | 被保险主体 |
| `📋` | 合同单据 | 投保单、保险单、批单、保险凭证 |
| `▣` | 第三方 | 公估人、受益人、代理人 |

## 标注规则

1. **保持原文完整**：只在实体周围添加标记，不改动文字
2. **同一实体多次标注**：文中每次出现都标注
3. **精准标注**：只标注明确的保险实体，不过度标注
4. **章节标题必标**：总则→〖§总则〗，保险责任→〖§保险责任〗
5. **条文编号必标**：第一条→〖¶第一条〗
6. **险种名称必标**：财产基本险（2025版）→〖@财产基本险（2025版）〗
7. **输出格式**：Markdown，用 # 和 ## 标记层级，段落间空行分隔
8. **不要添加解释**：直接输出标注后的文本

## 标注粒度指南

- ✅ 标注：火灾 → 〖⚡火灾〗（具体的承保风险）
- ❌ 不标：损失（太泛）
- ✅ 标注：如实告知 → 〖◈如实告知〗（明确的法律义务）
- ❌ 不标：应当（通用情态动词）
- ✅ 标注：投保单 → 〖📋投保单〗（具体的合同文件）
- ❌ 不标：书面形式（通用表述）
- ✅ 标注：货币赔偿 → 〖⛓货币赔偿〗（具体的赔付方式）
- ❌ 不标：赔偿（太泛）

## 险种名称对照（常见简称→全称）

保险条款中出现的险种名称都应用 `@` 标注全称：
- 财产基本险 → 〖@财产基本险（2025版）〗
- 财产综合险 → 〖@财产综合险（2025版）〗
- 财产一切险 → 〖@财产一切险（2025版）〗
- 机器损坏保险 → 〖@机器损坏保险（2025版）〗
- 营业中断保险 → 〖@营业中断保险（2025版）〗

## 示例

输入：
```
总则
第一条 本保险合同由保险条款、投保单、保险单以及批单组成。
保险标的
第三条 下列财产可作为保险标的：属于被保险人所有的建筑物、机器设备。
保险责任
第六条 由于火灾、爆炸、雷击造成保险标的的损失，保险人负责赔偿。
责任免除
第八条 地震、战争造成的损失，保险人不负责赔偿。
```

输出：
```
# 〖@财产基本险（2025版）〗

## 〖§总则〗

〖¶第一条〗 本保险合同由〖📋保险条款〗、〖📋投保单〗、〖📋保险单〗以及〖📋批单〗组成。

## 〖§保险标的〗

〖¶第三条〗 下列财产可作为〖☐保险标的〗：属于〖◆被保险人〗所有的〖☐建筑物〗、〖☐机器设备〗。

## 〖§保险责任〗

〖¶第六条〗 由于〖⚡火灾〗、〖⚡爆炸〗、〖⚡雷击〗造成〖☐保险标的〗的损失，〖◀保险人〗负责赔偿。

## 〖§责任免除〗

〖¶第八条〗 〖✕地震〗、〖✕战争〗造成的损失，〖◀保险人〗不负责赔偿。
```
